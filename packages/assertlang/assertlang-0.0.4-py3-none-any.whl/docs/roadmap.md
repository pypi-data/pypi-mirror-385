
# AssertLang Roadmap

## Phase 1 (MVP) — 2–3 months
- CLI + core verbs
- Python runner only
- Input/output tools
- Basic report

## Phase 2 (Alpha) — 4–6 months
- Add Go + Node runtimes
- 2–3–4–5–6 framework complete
- Extensibility API

## Phase 3 (Beta) — 6–9 months
- Marketplace launch
- Concurrency tools
- Security sandboxing

## Phase 4 (Production) — 9–12 months
- Rust runtime
- Cloud deployment option
- Full CI/CD integration
